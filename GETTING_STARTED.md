# 快速入门指南

欢迎使用 AI 文风助手！这是一个帮助你使用 AI 创作时保持个人写作风格的工具。

## 🚀 立即开始

### 1. 运行项目

```bash
cd /Users/didi/Desktop/writing-style-ai
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

### 2. 配置 API Key

首次使用需要配置 DeepSeek API Key：

1. 访问 [https://platform.deepseek.com](https://platform.deepseek.com) 注册账号
2. 在控制台获取 API Key（通常以 `sk-` 开头）
3. 在应用的"设置"页面输入 API Key
4. 保存配置

**注意**: API Key 仅存储在你的浏览器本地，不会上传到任何服务器。

### 3. 构建你的文风库

前往"文风库"页面，有两种方式添加内容：

#### 方式一：上传文档
- 点击"选择文件"按钮
- 选择 `.txt` 或 `.md` 格式的文章文件
- 支持批量上传多个文件

#### 方式二：直接粘贴
- 在文本框中粘贴你的文章内容
- 输入标题（可选）
- 点击"添加到文风库"

**建议**：添加 3-5 篇你过往的文章（总字数 10,000+ 字）以获得更好的效果。

### 4. 开始对话

前往"对话"页面：

1. 在输入框中描述你的写作需求
   - 例如："帮我写一篇关于时间管理的文章"
   - 或："改写这段话：……"

2. 点击"发送"按钮

3. AI 会根据你的文风库生成内容

4. 你可以继续追问，让 AI 调整内容

## 📖 功能说明

### 文风分析

系统会自动分析你的文章，提取以下特征：

- **关键词**: 你常用的词汇
- **常用短语**: 你习惯的表达方式
- **句子长度**: 你的句式习惯
- **语气风格**: 随意/正式/幽默/中性
- **标点习惯**: 你使用标点符号的偏好

### 文风润色

启用文风润色后（默认开启），AI 生成的内容会：

- 使用你常用的词汇
- 模仿你的句式结构
- 保持你的语气风格
- 遵循你的表达习惯

### 数据管理

在"设置"页面可以：

- **导出数据**: 下载你的文风库和对话记录（JSON 格式）
- **导入数据**: 从之前导出的文件恢复数据
- **清除数据**: 删除所有本地数据

## 🎨 界面说明

### 导航栏

- **对话**: AI 对话界面，主要工作区
- **文风库**: 管理你的文章和查看分析结果
- **设置**: 配置 API Key 和调整参数

### 对话页面

- 上方显示对话历史
- 下方是输入框
- 右上角有"清空对话"按钮
- 如果没有文风库，会显示提醒

### 文风库页面

分为三部分：

1. **添加内容**: 上传文件或粘贴内容
2. **已导入内容**: 查看和管理已添加的文章
3. **文风分析结果**: 查看你的写作风格统计

### 设置页面

四个部分：

1. **DeepSeek API 配置**: 输入和保存 API Key
2. **模型参数**: 调整 Temperature 和 Max Tokens
3. **文风润色**: 开启/关闭文风润色功能
4. **数据管理**: 导出/导入/清除数据

## 💡 使用技巧

### 1. 构建高质量文风库

- 选择你最满意的文章
- 确保文章风格一致
- 总字数建议 10,000 字以上
- 定期更新和补充

### 2. 优化 AI 输出

- 提供清晰具体的需求描述
- 可以要求 AI 调整语气或长度
- 使用"重新生成"来获得不同版本

### 3. 参数调整

在设置中可以调整模型参数：

- **Temperature (0-2)**:
  - 低值 (0.3-0.5): 更确定、一致
  - 中值 (0.7): 平衡（推荐）
  - 高值 (1.0-1.5): 更有创造性

- **Max Tokens (500-4000)**:
  - 控制每次回复的最大长度
  - 建议: 2000

## 🔧 故障排除

### API 调用失败

1. 检查 API Key 是否正确
2. 确认 API Key 有足够余额
3. 检查网络连接

### 文风分析结果不准确

1. 添加更多文章（至少 3 篇）
2. 确保文章内容完整
3. 点击"重新分析"按钮

### 数据丢失

- 浏览器清除缓存会删除数据
- 建议定期使用"导出数据"功能备份
